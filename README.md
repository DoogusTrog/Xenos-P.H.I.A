# Xenos-P.H.I.A (Program Hooking Intelligent Assistant)
A program hooking visual assistant that can answer in detail about the topic you are viewing, based on LLaMa 3.2V.
REQUIREMENTS: 
ollama,
psutil,
pyautogui,
pywin32,
and Pillow.
After installing dependencies, pull LLaMa3.2 Vision using ollama. (ollama pull llama3.2-vision)
NOTE: There are currently prompting issues I have not updated and will not until I make the tool more token efficient.
